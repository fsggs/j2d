# jQuery.j2d
jQuery.j2d is jQuery adaptation of [j2ds](https://github.com/SkanerSoft/J2ds) graphic lib.

## Documentation
[Русская документация](https://github.com/fsggs/jquery.j2d/wiki/ru_Home)

## Browser Support

Consequently, it supports the following natively:

* Chrome 29+
* Firefox 30+
